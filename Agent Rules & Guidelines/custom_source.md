# Custom Source — URL reference (app default)

These URLs are the **default data sources** for the app. New users see this list in Custom Sources; they can **delete** any entry (with a confirmation warning) or add their own.

- **Custom Source** = the **endpoint / data source** (each URL is where the app pulls job data from).
- **Identity** = the user’s profile (skills, locations, job types, transport, etc.). The app parses and shows **related jobs** from Custom Source URLs **according to the Identity profile** (matching jobs to the user’s preferences and context).

**OpenClaw / Telegram bot:** The OpenClaw or Telegram bot is expected to scrape the URLs listed in Custom Sources and use Identity (skills, preferred communes, job types, etc.) for relevance filtering and scoring. You can set an optional **API key** (key icon next to Custom Sources in the app) for the bot to use when calling this app’s API or a third-party scraping service; the key is stored with your identity and exposed to the bot via the OpenClaw feed.
